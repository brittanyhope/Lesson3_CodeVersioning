Highlight #1 Version Control

This is a system that records changes to a file. Everytime you modify a file, or create a new version, it is stored in case you want to revert to an older version. This is helpful especially if you deleted part of a file that you want later.

Highlight #2 Repository

In version control systems, a repository is a database that stores the commits made to a file. This is where the specific versions are stored after you push them. You can add new versions, or pull from the repository.

Highlight #3 Git

Github is a website that stores repositories. You use the git commands to push your file into your repository on GitHub, where you can view your commits and pull older versions.
